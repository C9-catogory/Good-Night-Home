# Good Night Home V10｜Three.js 递进式疗愈房间

V10 在 V9 的 3D 梦境房间基础上加入了“层层点灯”和更多关卡雏形。

## 新增内容

- 8 个可玩章节：
  1. 点灯房：空间锚点
  2. 玄关之门：物品地址法
  3. 床头恢复站：五感降载
  4. 书桌归档台：工作记忆外化
  5. 情绪云房：情绪命名与扩展
  6. 毯子角：内在部分安放
  7. 边界门：温柔边界
  8. 星湾：价值小行动
- 每完成一层，房间灯光更亮
- 新增物品：
  - 记忆星尘
  - 钥匙
  - 水杯
  - 耳塞
  - 待办本
- 新增容器：
  - 月光盒
  - 月光托盘
  - 床头恢复区
  - 书桌收件箱
- 新增输入：
  - 今晚一句话
  - 情绪三层选择
  - 边界句
  - 明天小行动
- 输出：
  - 层层点灯晚安卡

## 运行

```bash
python -m http.server 8000
```

打开：

```text
http://localhost:8000
```

## 注意

Three.js 使用 ES module 和 CDN importmap，建议不要直接双击 index.html，而是用本地服务器运行。
