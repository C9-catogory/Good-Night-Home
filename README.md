# Good Night Home V11｜稳定通关 · 收纳疗愈房间

生成/修复日期：2026-06-29

## 这版修复的核心问题

V10 的问题不是某一个按钮坏了，而是“游戏主循环”被拆散了：

- “下一章”可以绕过整理流程，导致关卡、通关和疗愈学习脱节。
- “晚安整合卡”会直接标记最终通关，导致还没完成星湾小行动也能结束。
- 第一章点灯后过早进入第二章，但第一章目标其实还包括“记忆星尘进入月光盒”。
- 书桌章节要求“把未完成写下”，但原代码只要放下待办本就完成，心理整合缺了一步。
- 物品收纳时的心理知识太少，没有把“物品有家 → 可控感 → 创伤/警报整合”串起来。

## V11 的稳定主循环

```text
看到房间 → 点灯 → 拾取物品 → 找到物品的家 → 学习对应心理知识
→ 完成当前小关 → 自动点亮下一章 → 最后生成晚安整合卡
```

## 可玩章节

1. 点灯房：点亮床头灯，把记忆星尘放进月光盒。
2. 玄关之门：钥匙进入月光托盘。
3. 床头恢复站：水杯和耳塞进入床头恢复区。
4. 书桌归档台：待办本进入收件箱，并写下一件今晚先不处理的事。
5. 情绪云房：命名情绪、下层担心和需要。
6. 毯子角：看见“小小云”这个内在部分。
7. 边界门：写下一句边界句。
8. 星湾：选择价值方向和明天小行动，生成晚安整合卡。

## 操作

- 鼠标拖动：轻微环绕房间。
- 滚轮：慢慢靠近/远离。
- 点击发光物：互动、拾取、放置。
- Tab：切换热点。
- “提示 / 继续”：不会跳关，只会在当前关完成时继续；未完成时给出下一步提示。

## 本地运行

因为使用 Three.js ES module 和 importmap，建议用本地服务器打开：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

如果部署到 Netlify，直接上传整个文件夹即可。
