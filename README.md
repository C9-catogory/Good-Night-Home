# Drawescape V5｜绘梦境

**Drawescape** 是一个把绘画理解为“信息整合 + 三维建模 + 相机成像 + 光影渲染 + 风格压缩”的可视化小游戏。

V5 的目标不是做静态教程，而是做一个更接近真正游戏的 WebGL 原型：用户可以旋转 3D 世界、保存视角截图、调整透视、观察明暗交界线、模拟褶皱和风水流体，并学习如何把感受翻译成线条、颜色、比例、明暗和构图。

## V5 新增重点

- **真实 WebGL 3D 实验室**：动态加载 Three.js，支持盒体、球、圆柱、圆锥、环、房间、人头、树、布片、飘带等几何体。
- **可保存视角**：保存当前相机 yaw / pitch / FOV / 高度和截图，并可再次载入。
- **3D 截图与场景 JSON 导出**：便于把学习过程保存成图像或继续开发。
- **透视学院**：一点 / 两点 / 三点 / 正交轴测；拖动消失点、视平线，在可调网格上绘画。
- **光影与褶皱实验室**：可调光源方向、光源高度、形体复杂度、褶皱张力和材质硬度；可视化明暗交界线。
- **人脸与风格化实验室**：转头、俯仰、动漫化、Q版化、辅助线、光影强度。
- **自然与流体实验室**：风、水、烟、树、云；理解自然物不是固定轮廓，而是力场、密度和运动证据。
- **调色画板**：从固有色生成亮部、暗部、环境反光、漫画强调色。
- **感受翻译器**：把安静、紧张、温暖、孤独、勇气、梦等感受翻译成视觉语言。

## 运行方式

### 本地运行

直接双击 `index.html` 可以运行大部分 Canvas 模块。  
WebGL 3D 模块需要加载 Three.js CDN；建议用 VS Code Live Server 或 GitHub Pages 打开。

### GitHub Pages

1. 新建 GitHub 仓库。
2. 上传 `index.html`, `styles.css`, `app.js`, `README.md`。
3. 进入 Settings → Pages。
4. Source 选择 `Deploy from a branch`。
5. 选择 `main` / root。
6. 等待生成网页地址。
