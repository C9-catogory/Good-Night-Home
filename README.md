# Good Night Home V9｜Three.js 心理尺度梦境房间

这是一个 Three.js / WebGL 的 3D 梦境房间样机，目标是把 Good Night Home 从 2D 平面 RPG 进一步转成“真实房间心理尺度”的温暖密室疗愈空间。

## 设计重点

- 真实房间尺度：4.2m × 3.2m × 2.6m
- 视角高度：约 1.35m，更像坐在床边或安静站在房间里
- 受限相机：不能高速移动，不能无限旋转，避免眩晕和过载
- 低饱和、暖光、月光、漂浮星尘
- 交互对象：
  - 床头灯
  - 床头抽屉
  - 月光盒
  - 月光托盘
  - 记忆碎片
  - 钥匙
  - 小小云
  - 温柔暂停门
- 输出：晚安整合卡

## 操作

- 鼠标拖动：轻微环绕视角
- 滚轮：慢慢靠近 / 远离
- 点击发光物：互动
- Tab：切换下一个热点
- 顶部“心理尺度”：显示心理尺度说明

## 本地预览

由于使用 ES module 和 CDN importmap，建议用本地服务器：

```bash
python -m http.server 8000
```

打开：

```text
http://localhost:8000
```
