光影与褶皱：把理论串成可画的步骤

明暗交界线不是一条“阴影边”，而是表面法线与光线关系变化的轨迹。褶皱也不是随机线，是固定点、重力、拉力和材质共同产生的结构。

明暗交界线口诀

先找体块 → 找光源 → 判断每个面的朝向 → 画朝光/背光过渡 → 再加投影与 AO。不要先涂黑，先问“这个面朝哪里”。

褶皱口诀

先找固定点 → 找拉力方向 → 判断材料软硬 → 主褶皱长而清楚，次褶皱短而碎。布不是线条，是受力后的曲面。

Human Geometry / Anime / Chibi

人脸与风格化：人也是几何，只是关系更复杂

用坐标点、轴线、体块和比例参数来画人脸。再把三维信息压缩成动漫/Q版/漫画语言。

人脸校准顺序

头骨大体块 → 中线/眉眼线 → 颧骨/鼻底/嘴线 → 耳朵连接点 → 光源与明暗交界线 → 风格压缩。

Nature / Fluid / Wind

自然与流体：画看不见的力

风、水、烟、云、树叶不是固定轮廓，而是运动、密度、重力和材料反应留下的证据。

自然物口诀

不要画“风本身”，画它改变了什么：头发弧线、衣摆方向、树叶角度、水面波纹、烟雾拉伸。

Palette / Color as Information

调色画板：颜色是情绪、光照和空间的压缩包

从固有色出发，生成亮部、暗部、互补色、环境反光和漫画化配色。

颜色转译

固有色告诉“这是什么”；光照色告诉“它在哪里”；环境色告诉“周围是什么”；饱和度和对比度告诉“这件事在情绪上有多重要”。

Feeling → Drawing Language

感受翻译器：把内心转成视觉语言

感受不是空泛的，它可以被翻译成线条速度、比例变形、颜色温度、明暗对比、构图重心和空间距离。
